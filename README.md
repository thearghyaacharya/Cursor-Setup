# Portfolio Project — Developer Environment Setup

## Overview
This repository was created as part of a portfolio project process. The goal was to set up a modern AI-assisted development environment, document the process, and push the result to GitHub — demonstrating the ability to follow technical instructions, troubleshoot independently, and ship.

---

## Tools Installed

### 1. Cursor IDE
- Downloaded and installed from [cursor.com](https://cursor.com/)
- Cursor is a code editor built on VS Code with native AI assistance for writing, editing, and debugging code

### 2. Claude Code (Cursor Extension)
- Installed via Cursor's Extensions panel (`Ctrl+Shift+X` → search "Claude Code")
- Logged in using my Anthropic/Claude account
- Claude Code enables AI pair programming directly inside the editor

### 3. Codex (Cursor Extension)
- Installed via Extensions panel → search "Codex"
- Logged in to activate
- Codex adds AI-powered code completion and generation capabilities
