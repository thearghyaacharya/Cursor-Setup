# LinkedIn Post Collection Guide

Manual collection protocol for LinkedIn posts. Follow this format for every post saved.

---

## File naming convention
`YYYY-MM-DD-short-title.md`

Example: `2025-03-15-mouse-trap-cold-email-pattern.md`

---

## Template (copy this for every post)

```
# [Post Title or First Line of Post]

**Author:** [Full Name]  
**Date:** [YYYY-MM-DD]  
**URL:** [LinkedIn post URL]  
**Engagement:** ~[X] likes / [X] comments (approximate at time of collection)  

---

## Post Content

[Paste full post text here]

---

## Why This Was Selected

[1-2 sentences on what makes this post high signal for the research]

## Key Insight

[One sentence: the single most useful takeaway from this post]
```

## How to find posts

1. Go to the expert's LinkedIn profile
2. Click "Posts" tab
3. Filter by "Top" to find highest-engagement posts
4. Also check "Recent" for anything from the last 3 months
5. Prioritize posts with data, frameworks, or specific tactics over opinion posts

---

## Quality bar

A post is worth collecting if it contains at least one of:
- A specific number or data point (reply rates, open rates, CTR)
- A named framework or system
- A specific tool recommendation with context
- A counterintuitive or non-obvious insight
- A real campaign example (even anonymized)

Generic "here are 5 tips" posts without evidence → skip.
